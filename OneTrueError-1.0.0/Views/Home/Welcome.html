Welcome

Congratulations to taking the first step to less painful exception handling.

Quick start

  1. Install one of our nuget packages in your application
  2. Create a new application by using top menu.
  3. Copy the appKey/sharedSecret to your application
  4. Generate an unhandled exception in your application to get it analyzed.
  5. View the beauty in this site.

More information

OneTrueError is based on a client/server architecture. The client packages detect and capture exceptions together with contextual information (such as thread info, app info etc). That information is then uploaded to our server for analysis.

To get started you need to download one of our nuget packages (search for "OneTrueError" in the package manager). Feel free to drop us an email if you need help to get started with nuget.

Then visit the application by clicking on the "Dashboard" drop down to get the correct appKey/sharedSecret.
